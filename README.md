# react-form-app (S3)

## Features - 產品功能
功能：

簡單的註冊表單提交頁面，具有獨立的按鈕元件
2 個輸入欄位：姓名、電子郵件
1 個提交按鈕
當點擊提交時，會將使用者輸入的內容顯示在下方的方框中。

## Installing - 專案安裝流程
:one: 開啟終端機, 複製此專案至本機電腦:
```
git clone https://github.com/Playpaper/react-form-app.git
```
:two: 進入存放此專案的資料夾
```
cd react-form-app
```
:three: 啟動伺服器
```
npm start
```
